# Y&Z Global Design System

## Direction

Premium logistics energy above the fold, followed by Notion-style business clarity. The hero is cinematic and asymmetric; content sections are quiet, structured and easy to scan.

## Color

- Primary dark: `#002A35`
- Accent yellow: `#FFDA00`
- White: `#FFFFFF`
- Light background: `#F7F7F2`
- Soft blue: `#6682C2`
- Border: `rgba(0, 42, 53, 0.12)`
- Muted text: `rgba(0, 42, 53, 0.68)`

## Typography

- Display: Barlow Condensed, 800
- Body: Helvetica, Arial, sans-serif
- Display copy uses compact line-height and restrained negative tracking.

## Components

- Buttons: flat, decisive, 12px radius maximum, yellow primary and outline secondary.
- Cards: 12–16px radius, border or subtle surface contrast, never nested.
- Process steps: numbered, ordered and connected only when sequence matters.
- Icons: Lucide React with consistent 20–24px sizing and 1.75–2px stroke.

## Motion

Expo-out easing `[0.16, 1, 0.3, 1]`. Animate only transform, opacity, scale, rotate and SVG path length. Respect reduced motion.

## Layout

Maximum content width 1440px with fluid side padding. Use asymmetric hero composition, generous section spacing and responsive one/two/three-column content patterns.
